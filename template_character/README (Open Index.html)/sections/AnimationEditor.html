Character Template ReadMe

Animation Editor

Now, Isaac doesn't actually directly import png files, instead it reads a special file called an anm2 file.

This file tells Isaac how the text should be animated and what not.

We will be using this tool associate our images we just made with the actual character now.

you can find this tool in the tools folder of the Isaac Directory.

My suggestion is to associate .anm2 files to this editor so its easier to open them in the future.

To do this you will need to open the editor in Administrator.

Go up to settings, and click the "Associate ..." line.

you can then close the application if you wish, and just go to the folder where the anm2 file is and open them directly.

if you do so, skip over this next line.

Once the editor is open, go up to File, and click Open. then you will need to go .\content\gfx

open charactermenu.anm2 On the right side of the program you will see the animations that exist within this anm2. what you need to do is click on Alpha, hit duplicate, and name it whatever you character's name is in players.xml (including caps) Then delete Alpha.

With your character selected on the left side you should see the bottom panel called the "timeline". when you click one of those yellow squares with a dot it will let you edit that frame of the animation. the only frame you Need to edit is the "Name" frame. just ensure your characters name is selected with the red bounding box.

Once your done save the file (Ctrl+S or File->Save). It will warn you about overwriting an existing file. hit accept. for your tainted character you will need to do the same thing but in the charactermenualt.anm2 file

This same process is repeated for characterportraits.anm2 and characterportraitsalt.anm2, only the frame there you need to edit is the Portrait.

Death Portraits anm2

Ok, now we just need to do the death screen anm2, so open death screen.anm2 Duplicate Alpha, renaming it your characters name. Then delete Alpha.

Now this is where it's a little different, the only thing you need to change in this file is the Name sprite. If you want to change other things go ahead of course, but I won't be covering that.

Once your done save it. For your tainted character you need to do the same thing, but in death screen alt.anm2.
