Character Template ReadMe

Metadata.xml

Your almost to the finish line!

by now you should have tried loading up your game to test your mod out. When you did it the first time you may have noticed that a metadata.xml file was generated.

If you haven't already what you need to do is delete that file, rename your character template folder to whatever name you want your mod to be named, and launch the game to genearted a new metadata.

Important to know: Loading order is entirely based on alphabetical ordering. Keep this in mind if your making use of another mod in your own.

This means that the higher on the list the mod is in the mod list in-game, it will load before the things lower than it. likewise, this means things lower in the list will always win if more than one mod is trying to load the same thing.

Now we need to open the ModUploader tool. You can find this tool inside the Isaac Directory.

once open click the "Choose Mod..." browse to your mod folder and choose the metadata.xml

now you can change everything to your hearts galore. don't hit upload or worry about the thumbnail image, or "change notes" section yet. once your done close the uploader, it will save what you did.
