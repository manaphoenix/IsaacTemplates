Character Template ReadMe

Welcome to Template Character Mod

This is the guide for the template that will take you step by step through what you need to do to create a character.

This guide is sorted into sections. Each section is not necessarily sorted in order they should or could be done, but each section does need to be completed.

To get started come up with a name for your character. Your tainted character should share the same name as your normal character.
